Lab 9 — React APIs (Recipe Search)

This lab is built with React.js and hosted on CodeSandbox. The app fetches recipes from the Spoonacular API and demonstrates working with public APIs in React.

Concepts demonstrated

  • useState — managing search query, fetched recipes, loading and error states.
  • useEffect — fetching initial recipes when the component first loads.
  • fetch API — calling the Spoonacular endpoints for recipe search and recipe details.
  • Event handling — search button click and Enter key listener.
  • React Router — navigation between Home, Recipe Details, and About pages.

Features

  • Search recipes by keyword (e.g. pasta, chicken).
  • View detailed recipe information including ingredients and instructions.
  • Recipe images displayed in a responsive grid layout.
